# Calculadora
Calculadora básica com uma pequena animação de entrada


Nesse projeto desenvolvi uma calculadora básica utilizando html/css e para as partes funcionais usei o javascript.

Html:
- Estrutura basica e os botões.
- Link do Javascript e o css.
- Divisão da calculadora em 4 blocos.

Css:
- Plano de fundo.
- Animação.
- Cores de botões e estrutura da calculadora.

Javascript:
- Funções de calculo.
- Validação caso esteja sem dados e solicite para imprimir algo.
- Função de backspace.
- Função de limpar tela da calculadora.
